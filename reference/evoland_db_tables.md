# Table active bindings for evoland_db

This file adds table active bindings to the `evoland_db` class using
R6's `$set()` method. These bindings provide read/write access to
database tables with automatic validation.

## Active Bindings Added

- `coords_t` - Coordinates table. See
  [`as_coords_t()`](https://ethzplus.github.io/evoland-plus/reference/coords_t.md)

- `periods_t` - Time periods table. See
  [`as_periods_t()`](https://ethzplus.github.io/evoland-plus/reference/periods_t.md)

- `lulc_meta_t` - Land use/land cover metadata. See
  [`as_lulc_meta_t()`](https://ethzplus.github.io/evoland-plus/reference/lulc_meta_t.md)

- `lulc_data_t` - Land use/land cover data. See
  [`as_lulc_data_t()`](https://ethzplus.github.io/evoland-plus/reference/lulc_data_t.md)

- `pred_data_t_float` - Float predictor data. See
  [`as_pred_data_t()`](https://ethzplus.github.io/evoland-plus/reference/pred_data_t.md)

- `pred_data_t_int` - Integer predictor data. See
  [`as_pred_data_t()`](https://ethzplus.github.io/evoland-plus/reference/pred_data_t.md)

- `pred_data_t_bool` - Boolean predictor data. See
  [`as_pred_data_t()`](https://ethzplus.github.io/evoland-plus/reference/pred_data_t.md)

- `pred_meta_t` - Predictor metadata. See
  [`as_pred_meta_t()`](https://ethzplus.github.io/evoland-plus/reference/pred_meta_t.md)

- `trans_meta_t` - Transition metadata. See
  [`as_trans_meta_t()`](https://ethzplus.github.io/evoland-plus/reference/trans_meta_t.md)

- `trans_preds_t` - Transition-predictor relationships. See
  [`as_trans_preds_t()`](https://ethzplus.github.io/evoland-plus/reference/trans_preds_t.md)

- `trans_rates_t` - Transition rates by period. See
  [`as_trans_rates_t()`](https://ethzplus.github.io/evoland-plus/reference/trans_rates_t.md)

- `intrv_meta_t` - Intervention metadata. See
  [`as_intrv_meta_t()`](https://ethzplus.github.io/evoland-plus/reference/intrv_meta_t.md)

- `intrv_masks_t` - Intervention masks. See
  [`as_intrv_masks_t()`](https://ethzplus.github.io/evoland-plus/reference/intrv_masks_t.md)

- `trans_models_t` - Transition models. See
  [`as_trans_models_t()`](https://ethzplus.github.io/evoland-plus/reference/trans_models_t.md)

- `alloc_params_t` - Allocation parameters. See
  [`as_alloc_params_t()`](https://ethzplus.github.io/evoland-plus/reference/alloc_params_t.md)

- `neighbors_t` - Neighbor relationships. See
  [`as_neighbors_t()`](https://ethzplus.github.io/evoland-plus/reference/neighbors_t.md)
